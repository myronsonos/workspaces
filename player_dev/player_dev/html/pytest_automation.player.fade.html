seek(unit, target)[source]¶

Repeatedly seek to the given location in a currently playing track :param str unit: unit to seek [TRACK_NR, REL_TIME, TIME_DELTA]

TRACK_NR = track number in queue REL_TIME = relative time to seek in current track TIME_DELTA = positive (forward) or negative (backwards) time to

seek from current position.
Parameters:target (str) – target to seek to (based on unit) if unit == TRACK_NR, target is the track number to seek to if unit == REL_TIME, target is a time string (ex: ‘00:25:45’, seek to 25:45 in track) if unit == TIME_DELTA, target is +/-time string (ex: ‘+00:00:30’ seeks 30 seconds forward from current position)
