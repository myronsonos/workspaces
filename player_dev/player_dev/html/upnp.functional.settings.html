NOTE!!! This test applicable to devices that have a serial console attached. In the event that all test bed devices have a serial console, this test can be enabled to run on that test bed.

Removes existing (if any) LogToExternal keyword from file and saves the LogToExternal key word value from the localsettings.txt dictionary.

Appends LogToExternal: [serial:0] to the localsettings.txt file. Bounces and waits for anacapa, retrieves the anacapa trace file and isolates the contents of that file from the last reboot. Searches for the diagnostic to determine if the persistent log outputs to the serial console.

Before completing the test, the saved LogToExternal key word (if any) is restored to the original value prior to test. :param zp: Device object :type other_zp: SonosZoneComponent

test_udp(**kwargs)[source]¶

Removes existing (if any) LogToExternal keyword from file and saves the LogToExternal key word value from the localsettings.txt dictionary.

Appends LogToExternal: [udp:<ip>:<port>] to the localsettings.txt file where the <ip> and <port> is the remote host address and destination port the device will forward the log messages as UDP datagrams.

Bounces and waits for anacapa, retrieves the anacapa trace file and isolates the contents of that file from the last reboot. Searches for the diagnostic to determine if the device is forwarding the logging message to the specified value.

Confirms via the WaitOnUDPLogging class the arrival of any logging messages from the test device.

Before completing the test, the saved LogToExternal key word (if any) is restored to the original value prior to test.

Applicable to zone player devices (not available on WD100 and bridges)

Parameters:zp – Device object
