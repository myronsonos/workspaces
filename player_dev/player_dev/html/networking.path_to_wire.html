test_advertise_wire_off_chan(**kwargs)[source]¶

Test to verify that a station mode zp will not include any wire path ie in its probes. When a useful wire connection is added to the zp, it will transition to sonosnet mode and will advertise first wire if the zp is a node. A non-node capable ZP (or leaf) will not advertise first wire in its probes regardless of its own ethernet carrier state. AP channel is not the same as the sonosnet HH channel.

Parameters:
  • zp (SonosZoneComponent) – zone player
  • advertises_first_wire (bool) – if zp will advertise first wire in its probes when it has carrier.
