Bases: upnp.functional.hls.hls_base.HLSBaseFixture

Conduct HLS bit rate adaptation tests. Based on available network bandwidth, the player selects the appropriate media playlist that can support the play back. Note that the initial implementation used the log inspector to confirm the player was opening (reading) the correct files that supported the available bandwidth. The problem with using the log inspector in this particular test is that it provided false positives/negatives since it was reading the contents of the anacapa trace which may have had the expected diagnostics from the current test or previous incarnations of tests which utilized the same playlists and just the right logging level. The UDP logger is more suitable to this test since it instantiates a logging session that is unique to the test. Addresses SWPBL-46155 The script tests on specific players since not all players can switch to higher bit rates (when that rate is available) due to SWPBL-46264.
