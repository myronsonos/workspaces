class other.vlc.vlc.VLCInterface(server_ip, server_port)[source]¶

Bases: object

Control a running VLC Server through the HTTP interface

To start VLC in headless mode with the VLC interface, do:

./VLC -I http

Make sure that ‘plugins/shared/lua/http/.hosts’ has the final two lines uncomments so you can access the VLC server from outside of the host machine.
