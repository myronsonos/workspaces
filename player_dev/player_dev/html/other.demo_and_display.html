other.demo_and_display.3_1_setup_verification module¶

This script should be run after the BrightSign has configured the testbed. This verifies that: - The household ID is set on all devices - The PLAYBAR is set to play toslink input - The SUB is bonded to the PLAYBAR - All players are set to the defined starting volume - All player mute states are set correctly

other.demo_and_display.setup_testbed module¶

This script should be used to reset, or setup, a BrightSign testbed. This will find all devices on the network and verify that they exactly match the expected device list from the config file. It then factory resets each device and verifies that telnet is still locked once completed.

Run this script before powering a BrightSign on, or before cycling the BrightSign power.
