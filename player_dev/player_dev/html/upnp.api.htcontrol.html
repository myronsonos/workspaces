upnp.api.htcontrol.test_ht_control_events module¶

upnp.api.htcontrol.test_ht_control_events.CONFIG_FILE = 'ht_control.json'¶
The HTControl service has four state variables:
IRRepeaterState, LEDFeedbackState, RemoteConfigured, TOSLinkConnected
Notes:
  1. LEDFeedbackState is listed in DeviceSpy as a state variable. However, it is not readable via HTControl.get_state_variable(“LEDFeedbackState”) and times out. A grep of python code did not find any occurences. Therefore this test does nothing with LEDFeedbackState.
  2. RemoteConfigured is listed in DeviceSpy as a state variable. However, it is not readable via HTControl.get_state_variable(“RemoteConfigured”) and times out. A grep of python code only shows it as a return value to the HTControl.IsRemoteConfigured() function call. Therefore this test does nothing with RemoteConfigured.
