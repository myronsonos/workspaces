voice.mips_trending module¶

This module contains a MIPS performance trending utility to track MIPS (cpu% utilization) performance for each block and for the overall voice capture class given two versions on a particular branch. this utility will webcrawl results on the camb-sqa-nas2 server and use the JSON files within the server to create trending plots. The functionality is slightly similar to how we plot CPU results (see tests/src/other/cpu_trending2.py)
