The GroupManagement service has four state variables:
GroupCoordinatorIsLocal, LocalGroupUUID, ResetVolumeAfter, VolumeAVTransportURI
Notes:
  1. ResetVolumeAfter is listed in DeviceSpy as a state variable. However, it is not readable via GroupManagement.get_state_variable(“ResetVolumeAfter”) and times out. A grep of python code only shows it as an arg. Therefore this test does nothing with ResetVolumeAfter.
  2. VolumeAVTransportURI is listed in DeviceSpy as a state variable. However, it is not readable via GroupManagement.get_state_variable(“VolumeAVTransportURI”) and times out. A grep of python code only shows it as an arg. Therefore this test does nothing with VolumeAVTransportURI.
