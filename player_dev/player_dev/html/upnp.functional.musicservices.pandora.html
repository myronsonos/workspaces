Bases: sonos.workflow.fixture.WorkflowTestFixture

Confirm the player determines the track duration from a Pandora MP3 file. Some recent MP3 encodings on Pandora caused the player not to be able to determine the track duration length. This test partially verifies SWBPL-67279 to make sure the player is able to determine the duration on a track from “captured” Pandora track instances which exhibited the symptom.
