upnp.functional.events.evented_position_not_greater_than_duration module¶

Regression test for PTS-2373 - verify that the player never reports the current track position as > track duration.

These tests verify that the player never over reports the current track position beyond the length of the track. Before this bug was fixed it was possible for the player to report a track’s current position beyond the actual track duration provided in the metadata.

https://jira.sonos.com/browse/PTS-2373
