pytest_automation.networking.test_station_mode_thrput_smoke module¶

Pytest to run a quick smoke test throughput of a station mode DUT

This test is intended to be on the cb-wifi-x testbeds which have exactly 1 ZP DUT (mediatek radio based) and 1 openWRT AP which are both in separate iso chambers.
