is_surround_mode_enabled(zp)[source]¶

Get content in file in /jffs/localsettings.txt there should be a line like the following EQ: [AMV25 LV100 RV100 B0 T0 LY FN SS6 LEV0 SWY SC0 SP0 DL0 SL0

AD0 NMN LRD1 RRD1 SRY SSM1]

to get the current surround sound mode, we need SSM1 to show it is enabled or SSM0 to show it is disabled
