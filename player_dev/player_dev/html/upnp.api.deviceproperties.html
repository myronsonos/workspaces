upnp.api.deviceproperties.test_device_properties_events.ENABLED = '1'¶
The DeviceProperties service has twenty-five state variables:

AutoplayIncludeLinkedZones,AutoplayRoomUUID, AutoplayUseVolume, AutoplayVolume, ChannelMapSet, Configuration, CopyrightInfo, DisplaySoftwareVersion,ExtraInfo,HardwareVersion,HouseholdID, HTAudioIn,HTFreq, HTSatChanMapSet,Icon,IPAddress, IsZoneBridge,LEDState,MACAddress,SatRoomUUID,SerialNumber, SettingsReplicationState,SoftwareVersion, ZoneName

SettingsReplicationState,,

Notes:
  1. SatRoomUUID is listed in DeviceSpy as a state variable. However, it is not readable via DeviceProperties.get_state_variable(‘SatRoomUUID’) and times out. A grep of both python and C++ code only shows it as an argument to the RemoveHTSatellite() function call. Therefore this test does nothing with SatRoomUUID.
  2. HTFreq is only readable via a Playbar. It always returns 0. I borrowed a piece of script from channel_change_2ghz.py and can see the channel change to the specified channel/frequency. I can see what appears to be correct changes in dmesg. However, HTFreq does not change on the S9 and is not readable on the S1. e.g. (166261)sat: enable=1 (166262)sat: saw primary on 2412 (htchan=5180) (166262)sat: freq=5180 (was 2412) 1

Test Cases: - Set/Get LEDState and compare to State variable LEDState - Set/Get HouseholdID and compare to State variable HouseholdID - Set/Get ZoneAttributes and compare to State variables ZoneName,

Configuration,and Icon.
  • Set/Get AutoplayUseVolume and compare to State variable AutoplayUseVolume
  • Set/Get AutoplayVolume and compare to State variable AutoplayVolume
  • Set/Get AutoplayLinkedZones and compare to State variable AutoplayLinkedZones
  • Set/Get AutoplayRoomUUID and compare to State variable to AutoplayRoomUUID
  • Add/Remove BondedZones and compare to State variable ChannelMapSet
  • Create/Separate StereoPair and compare to State variable ChannelMapSet
  • Add/Remove HTSatellite and compare to State variable HTSatChanMapSet
  • GetZoneInfo and compare to State variables CopyrightInfo,DisplaySoftwareVersion, ExtraInfo,HardwareVersion,HTAudioIn,IPAddress,MACAddress,SerialNumber,SoftwareVersion
  • Get all devices and compare zp.is_bridge() result to State variable IsZoneBridge
  • Get initial SettingsReplicationState variable. Change a system settings variable. Wait for settings to match. Compare Initial/Updated/Final result to State variable SettingsReplicationState.
