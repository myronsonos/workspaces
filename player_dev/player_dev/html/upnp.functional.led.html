test_leds(**kwargs)[source]¶

Tests the voice related LED states by playing a voice tap and checking anacapa.trace for the correct LED bitmask settings. If the near_field parameter is set to True, it will also test near field but only does so with no voice command given. The reason for only testing silence with near field is that voice tap playback is triggered by unmuting the microphone. As a result, attempting to trigger near field with a voice tap would result in the same behavior as testing far field with a voice tap.

Parameters:
  • zp (SonosZoneComponent) – The DUT
  • muted (bool) – Whether or not the DUT should be muted to start the test
  • led_on (bool) – Whether or not the DUT should have the status LED on
  • led_pattern (list of str) – The pattern of LED bit masks to watch for
  • tap (str) – The voice tap file name
  • near_field (bool) – Whether or not to test near field
