other.cpu.cpu_confluence_display module¶

Module for graphing CPU results sample data from web interface.

Example URL http://camb-sqa-nas2/AutomationServer/PlayerDev/33.7-28170/cpu/samples/ where:

branch: PlayerDev version: 33.7-28170

An example “raw data” file is: http://camb-sqa-nas2/AutomationServer/PlayerDev/33.7-28170/cpu/samples/CPU-library-AAC-standalone-40684-S1-LF,RF-samples.xml where:

CPU-<resource>-<configuration>-40684-<zp-channel>-samples.xml
Note:
Not all resources and configurations are available for all versions. Phase 1: As it is loading raw samples via http, it can be slow. Phase 1.1: Include GroupBy(branch) Phase 2: Put all raw samples into a persistent highly-available database suited for massive queries. Phase 3: Use aggregation functionality if possible from DB.
