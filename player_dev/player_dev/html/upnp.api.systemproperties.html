Edit metadata for this service account.

EditAccountMd is used to modify the metadata section of the specified service in accounts.xml. This is used to set things like:

  • Rhapsody cobrand ID
  • Last.FM scrobbling state

Arguments: service_id – integer, ID of added service account to edit username – string, username associated with added service account updated_metadata – string, updated metadata for service account
