Exercise the AlarmClock.GetHouseholdTimeAtStamp() function across the supported timezone indexes, 0-74, and
verify that the returned results contain proper values. This is a device-to-device only call. It does not
get called by client code.

The stamp comes from the TimeSource Stamp element from the contents of /jffs/settings/alarmclock.xml.

<AlarmClock LastUpdateDevice="RINCON_000E58F9C12601400" Version="4">
    <Alarms LastUpdateDevice="RINCON_000E58F9C12601400" Version="0">
    </Alarms>
    <Format Time="INV" Date="INV"/>
    <TimeSource Stamp="RINCON_000E58F9C12601400:3"/>
    <TimeZone Index="4" AutoAdjustDst="0"/>
</AlarmClock>

The following is the Function description from the AlarmClock service.

        Return this timestamp as the device would display it.

        Exceptions:
        UPNP_ERROR_801_TIME_NOT_AVAILABLE

        Returns:
        string -- timestamp with device date format and time zone

"""

    def _get_alarm_clock_time_stamp(self, device):
        '''
        Return the timestamp from /jffs/settings/alarmclock.xml.
        The TimeSource Stamp is not returned via the server. Reading the url data or
        the file text into an XMLAccessor had formatting issues. This was the simplest
        way to get the required time stamp field.
        '''

        alarm_clock_settings = None
        res = device.cli.command("cat /jffs/settings/alarmclock.xml")
        res_file = res.split('\r\n#')[0]
        rind = res_file.find("TimeSource Stamp=") + len("TimeSource Stamp=")
        if rind:
            stamp = res_file[rind:]
            if stamp:
                end = stamp.find("/")
                if end:
                    alarm_clock_settings = stamp[:end].replace('"', '')

        return alarm_clock_settings

    def _compare_date_time(self, dt1, dt2):
        '''
        Date should match. Hours and minutes should match, Seconds role away.
        An expected time may also have a 59 count. That may roll up others as well.
        '''
        result = False
        sdt1 = dt1.split(":")
        sdt2 = dt2.split(":")
        if sdt1[0] == sdt2[0]:
            if sdt1[1] == sdt2[1]:
                return True
        if sdt1[2] == "59":
            dt = datetime.datetime.strptime(dt1, "%Y-%m-%d %H:%M:%S")
            td = dt + datetime.timedelta(0, 3)
            result = self._compare_date_time(str(td), dt2)
            if not result:
                self.logger.debug("Result of nested datetime compare was False. td: %s; dt2: %s" % (str(td), dt2))
        return result
