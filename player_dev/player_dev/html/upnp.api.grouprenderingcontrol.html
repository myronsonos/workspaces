Determines whether devices are available on a given testbed to run tests against. If a device is available when a test calls on this method, it will pick it from a list and return it to test against. Wait for the snap shot group volume to converge on the group coordinator (GC) play zone. It monitors, using UDP log, a specific diagnostic the GC device will generate when all the group members have completed the group rendering registration with the GC.
